# Search Relevance Guide

If users report that knowledge-base answers cite unrelated articles, compare query intent, top retrieved chunks, and citation support before changing the prompt.

Prefer improving retrieval data first: add missing headings, split oversized sections, and include operational terms that support the user's question. Prompt changes are secondary when the wrong document is selected.

Track top-k source hit rate and nDCG after each retrieval change.
